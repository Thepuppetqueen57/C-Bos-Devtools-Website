C-Bos Devtools

C-Bos Devtools is a tool for developers who look to help with C-Bos. And for me lol. But basically C-Bos Devtools has some utilities that might be helpful for working on C-Bos

To install C-Bos Devtools make sure you have nodejs installed. Then go to your terminal and run install -g cbos-devtools then run cbosdev help.
